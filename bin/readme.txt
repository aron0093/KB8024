/bin

-Scripts to run models for projects. Also contains the modules required to run these scripts. Listed below are the user scripts.

    -SignalP predict: Choose to use models generated for the SignalP project. You can test against 50 TOPDB proteins or get predictions for a fasta file. Predictor can process multiple inputs at the same time. This took time, please appreciate it.
    -Predictor: This script is my final submission for the course. Run to get predictions for proteins stored in a fasta format file.
